* 문제 풀이 방식과 벌금 기준
 - 기초 > 연습 > 문제 순서로 문제 풀이
   (같은 table 안에서 어떤 문제를 풀지는 자유)
 - 하루에 최소 1개의 문제를 풀고 git과 개인 블로그에 올릴 것
 - 일주일에 이틀 휴식 가능하며, 3일 이상 휴식 시 초과한 일 수 x 1000만큼 벌금
 - 벌금은 상대방에게 바로 카카오페이/토스로 송금
 - 일주일 기준은 월요일~일요일
 - 일주일이 지나면 벌금 정산 후, 카카오톡으로 벌금 공지 예정

* git 파일 관리
 - git은 public으로 만들기
 - root 폴더에 study_algo_record_date.xlsx, rule.txt, 기초(폴더), 연습(폴더), 문제(폴더) 넣어 놓기
 - study_algo_record_date.xlsx 표 제목에 기초하여 각 폴더 안에 폴더 추가 후, 그 안에 문제 풀이 코드 올리기
   예) 기초 > 수학, 브루트 포스, 브루트 포스 - N과 M ...
